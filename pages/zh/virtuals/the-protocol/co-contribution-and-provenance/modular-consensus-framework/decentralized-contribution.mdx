# 去中心化贡献

去中心化贡献是我们生态系统的一个基本要素，它允许外部贡献者通过增强人工智能智能体的能力，助力推动指数级增长。贡献者可以改进智能体功能的各个方面，成功的贡献将被铸造成非同质化代币（NFT）并转移给贡献者。这既作为贡献的证明，又便于进行奖励分配。

为方便贡献者通过我们的平台轻松提交他们的模型或数据集，贡献流程已被简化。一旦提交，会发生以下操作：

![](/images/virtuals/decentralized-contribution.avif)

#### 1. 带有元数据文件的贡献 NFT 创建

对于每一项贡献，都会铸造一个 NFT，其中包含有关该贡献的详细元数据（例如描述、版本、类型）。这个 NFT 会自动发布到星际文件系统（IPFS）上，以确保贡献细节的去中心化和永久存储。

#### 2. 智能合约交互

从提交到 NFT 铸造的整个过程都由智能合约管理。这些智能合约负责在链上验证贡献，确保所有贡献都在区块链上被记录和追踪。

#### 3. 所有权和访问权

拥有贡献 NFT 赋予贡献者某些权利，包括对作品的控制权以及由此产生的任何奖励。将 NFT 转让给另一方，也会转让这些所有权权利，包括领取与该贡献相关的奖励或激励的能力。

贡献者可以通过我们的平台提交他们的贡献。
