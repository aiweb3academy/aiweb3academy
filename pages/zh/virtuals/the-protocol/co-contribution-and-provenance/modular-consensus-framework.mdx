# 模块化共识框架

模块化共识框架为不同的利益相关者规范并简化了各种流程：
- **贡献者**：
  - **贡献流程**：贡献者通过我们的前端，利用模块化共识框架提交他们的提案。无论提案是否被接受，每个提案都会生成一个贡献非同质化代币（NFT），以此验证提交的来源。
  - **不可变贡献库中的状态终结性**：被接受的贡献会在链上铸造成服务 NFT，并分配到不可变贡献库（ICV）内相应的 Virtual 地址，这验证了它们已融入 Virtual 生态系统。
- **验证者**：
  - **策略与资源分配**：流动性提供者通过对特定智能体进行质押，来决定协议的战略方向，根据质押权重影响去中心化自治组织（DAO）的资源分配。
  - **验证与最终确定**：利用委托权益证明机制，代币持有者将代币委托给合格的验证者，这些验证者负责确定每个智能体的最终状态。
