# IP 智能体与功能性智能体

大多数 Virtuals 智能体都是 IP 智能体，代表着某种个性或角色，就像一个虚拟的存在体。例子包括一只青蛙、一个网络梗图形象、唐纳德·特朗普、泰勒·斯威夫特、约翰·维克、鸣人或者史酷比。话虽如此，在幕后，核心贡献者们正在开发多种功能性智能体，以提升终端用户与人工智能虚拟角色互动时的整体体验，同时确保这些虚拟角色能够无缝融入虚拟世界。我们将这些称为功能性智能体。
